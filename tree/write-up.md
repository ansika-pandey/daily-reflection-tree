# Design Explanation

This reflection tree is designed to guide users through three psychological axes: locus of control, contribution, and perspective.

For Axis 1 (Locus of Control), I created questions to identify whether the user perceives events as within their control or influenced by external factors. The goal was to surface awareness of personal agency.

For Axis 2 (Contribution vs Entitlement), I focused on understanding whether the user contributed value or expected outcomes from others. The questions highlight actions versus expectations.

For Axis 3 (Radius of Concern), I expanded the reflection to include others, encouraging the user to think beyond themselves and consider team, colleagues, and customers.

The branching structure is deterministic, meaning each answer leads to a predefined path. Fixed options were used to avoid ambiguity and ensure consistent outputs.

If given more time, I would expand the tree with more nuanced scenarios, deeper reflections, and better transitions between nodes.
