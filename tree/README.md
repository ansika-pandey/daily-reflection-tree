# Daily Reflection Tree

This project is a deterministic reflection system built using a structured decision tree.

## Overview
The system guides users through a structured end-of-day reflection based on three psychological axes:

1. Locus of Control (Victim vs Victor)
2. Contribution vs Entitlement
3. Radius of Concern (Self vs Others)

## Features
- Fully deterministic decision tree
- No AI/LLM used at runtime
- Fixed options for all questions
- Structured reflections based on user choices

## Files
- tree/reflection-tree.tsv → main decision tree
- write-up.md → design explanation

## Purpose
This project demonstrates how human psychological concepts can be converted into structured, deterministic systems.
